Displays a category posts carousel.

== Description ==

This plugin will allow your site to display your category posts into a jquery carousel which can be added to your website's sidebar.



== Installation ==

1. Unzip and upload `posts_carousel` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add Posts Carousel Widget to any of your sidebar and fill in your desired values
4. Go to Settings->Post Carousel and configure

== Frequently Asked Questions ==

= Can I display multiple category? =

No. The current version only supports one category.

= Can I adjust the speed of the carousel? =

Yes, you can adjust the carousel's speed on Settings->Posts Carousel Page.
